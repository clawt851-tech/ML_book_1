# 第 19 章 市场微结构特征

## 19.1 写作动机

订单簿和成交流中蕴含**预测信号**——这些信号对应市场微结构经济学的具体机制。本章按"代际"展示主要微结构特征。

## 19.2 文献综述

市场微结构理论关注:做市商如何从订单流推断信息;知情交易者如何隐藏意图。三代模型对应三类特征。

## 19.3 第一代: 价格序列

### 19.3.1 Tick Rule
$b_t = \mathrm{sign}(\Delta p_t)$,零变动时沿用上一值。简单但极有用。

### 19.3.2 Roll Model

无信息交易下的隐含半价差:

$$c = \sqrt{-\mathrm{Cov}(\Delta p_t, \Delta p_{t-1})}$$

只在 Cov < 0 时有效(否则模型不适用)。

### 19.3.3 High-Low 波动率估计 (Parkinson 1980)

$$\hat{\sigma}^2 = \frac{1}{4 \log 2}\,E[(\log H - \log L)^2]$$

比仅用 close-to-close 高效。

### 19.3.4 Corwin-Schultz (2012) 价差估计

仅用 (high, low) 估计买卖价差:

$$\bar{S} = 2(e^{\alpha} - 1)/(1 + e^{\alpha})$$

其中 $\alpha$ 由两日 $\beta$ 和 $\gamma$ 计算。

## 19.4 第二代: 战略交易模型

### 19.4.1 Kyle's Lambda (价格冲击)

回归 $\Delta p_t = \lambda \cdot b_t v_t + \varepsilon_t$,$\lambda$ = 单位有符号成交量造成的价格变动。$\lambda$ 越大 = 流动性越差。

### 19.4.2 Amihud's Lambda (流动性)

$$\mathrm{ILLIQ}_t = \frac{|r_t|}{V_t \cdot p_t}$$

每美元成交带来的价格变动绝对值。Amihud (2002) 经典指标。

### 19.4.3 Hasbrouck's Lambda

回归 $\log r_t = \lambda \cdot \mathrm{sign}(\Delta p_t)\sqrt{|p_t v_t|} + \varepsilon$。Hasbrouck (2009) 改进的价格冲击度量。

## 19.5 第三代: 序列交易模型

### 19.5.1 知情交易概率 (PIN)

Easley et al. 系列工作。基于"无知情订单率 + 知情订单率" 的混合模型。

### 19.5.2 VPIN (Volume-Synchronized PIN) ★

Easley, López de Prado, O'Hara (2011) 提出。**不依赖估计未观测变量**:

1. 用**成交量时钟**(Volume Clock)而不是时间;每桶累积固定成交量。
2. 用 Bulk Volume Classification (BVC):基于标准化收益率的 CDF 把成交量分到买/卖方:
   - $V^B_t = V_t \cdot \Phi(z_t)$, $V^S_t = V_t \cdot (1 - \Phi(z_t))$
3. 在最后 $n$ 桶上计算:

$$\mathrm{VPIN} = \frac{\sum_t |V^B_t - V^S_t|}{\sum_t V_t}$$

VPIN 在 2010 闪崩前已显著上升——是闪崩的领先指标(用于 SEC 监管)。

## 19.6 微结构数据集额外特征

| 特征 | 说明 |
|------|------|
| 订单大小分布 | 大单分布偏离正态 = 知情交易迹象 |
| 取消率 / 限价单 / 市价单比例 | 市场情绪指示器 |
| TWAP 算法签名 | 大型机构执行的足迹 |
| 期权市场 | 隐含波动率偏度、风险逆转 |
| 有符号订单流的序列相关 | 大单切片留下的 "runs" |

## 19.7 什么是微结构信息?

简短回答:任何可以从订单簿/成交记录推断**信息不对称**的特征都是微结构信息。

## 19.9 本章核心要点

1. 三代微结构特征:价格序列 → 战略模型 → 序列模型。
2. **VPIN** 是 ML 特征工程的明星——闪崩、流动性危机的领先指标。
3. tick rule + roll model 是任何 ML 流水线的基础特征。
4. Amihud 流动性、Kyle 价格冲击是因子投资的关键变量。
