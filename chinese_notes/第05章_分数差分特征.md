# 第 5 章 分数差分特征

## 5.1 写作动机

金融时间序列由于套利约束,信噪比极低 (López de Prado [2015])。雪上加霜的是,标准平稳化变换(整数差分)会**进一步抹掉信号中的记忆成分**。

价格序列通常**非平稳但有记忆**;收益率(整数差分一阶)平稳但**没有记忆**(过了一个采样窗口就完全忘掉历史)。

**两难:** 推断分析需要平稳序列,但**记忆**才是预测能力的基础。本章引入分数差分,在保留尽可能多记忆的前提下使序列平稳。

## 5.2 平稳性 vs 记忆的两难

- 协整方法的重要性恰在于"建模带记忆的序列"。
- 0 差分(原序列)和 1 差分(收益率)只是两个极端,中间存在广阔的"分数差分"区域。
- 监督学习需要平稳特征(把新样本映射到训练样本),但平稳性是必要非充分条件。
- 整数差分过度差分,丢掉信号。

## 5.3 文献综述

分数差分至少可追溯到 Hosking [1981],他推广了 ARIMA 模型,允许差分阶数为分数,从而获得长期持续性和反持续性。但此后该方向只有约 8 位作者总共 9 篇论文,大多技术性强难落地。

## 5.4 方法

### 后移算子 $B$ 的二项式展开

$$(1 - B)^d = \sum_{k=0}^{\infty} \binom{d}{k}(-B)^k = 1 - dB + \frac{d(d-1)}{2!}B^2 - \frac{d(d-1)(d-2)}{3!}B^3 + \cdots$$

应用到时间序列:

$$\tilde{X}_t = \sum_{k=0}^{\infty} \omega_k X_{t-k}, \quad \omega = \left\{1, -d, \frac{d(d-1)}{2!}, \dots, (-1)^k \prod_{i=0}^{k-1}\frac{d-i}{k-i}, \dots\right\}$$

### 5.4.1 长记忆

权重序列具有递推关系:

$$\omega_k = -\omega_{k-1}\,\frac{d - k + 1}{k}, \quad \omega_0 = 1$$

- $d \in (0, 1)$:权重快速衰减但**保留长记忆**。
- $d = 1$:只有 $\omega_0 = 1, \omega_1 = -1$,其余为 0(就是一阶差分,记忆完全丢失)。

## 5.5 实现

### 5.5.1 扩展窗口 (Expanding Window)
Snippet 5.2 用所有可用历史计算 $\tilde{X}_t$。问题:权重剖面随历史长度变化,不同时间点 $\tilde{X}_t$ 的统计性质不一致。

### 5.5.2 固定宽度窗口 (Fixed-Width Window FFD) ★

只保留 $|\omega_k| > \tau$ 的项,忽略尾部小权重。这样所有 $\tilde{X}_t$ 用相同长度的权重,**统计性质平稳**。

```python
def getWeights_FFD(d, thres):
    w, k = [1.], 1
    while True:
        w_ = -w[-1]/k*(d-k+1)
        if abs(w_) < thres: break
        w.append(w_); k += 1
    return np.array(w[::-1]).reshape(-1, 1)
```

## 5.6 在最大记忆保留下达成平稳

实战流程:
1. 选阈值 $\tau$(如 $10^{-5}$),固定 FFD 窗口宽度。
2. 对一批 $d \in [0, 1]$ 计算 FFD 序列。
3. 对每个 $d$ 跑 ADF 平稳性检验。
4. 取**通过 ADF 检验的最小 $d^*$**。
5. 此时 $\tilde{X}_t$ 与原序列 $X_t$ 的相关系数仍可达 0.99 以上(对 E-mini SP500),保留了几乎全部记忆。

经验观察:大多数金融 dollar bar 序列在 $d^* \in [0.3, 0.5]$ 即可平稳——比 $d=1$ 保留多得多的信息。

## 5.7 结论

- 平稳化不必采用整数差分。
- FFD 让序列平稳但**仍带记忆**,显著提升 ML 模型的预测能力。
- 这是本书最重要的实战工具之一(常被独立引用)。

## 5.9 本章核心要点

1. 整数差分(返回率)抹掉记忆;0 差分(价格)不平稳。
2. **分数差分**在两端之间提供完整谱系。
3. 用 **FFD**(固定宽度)保证统计性质平稳。
4. 实战:选最小通过 ADF 的 $d$,可保留与原序列 0.99+ 相关。
5. 本章方法对 ML 模型预测能力提升显著。
