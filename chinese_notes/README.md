## 全书核心思想

### 三大回测定律 (Marcos's Three Laws of Backtesting)

1. **第一定律**: 回测不是研究工具,**特征重要性**才是。
2. **第二定律**: 一边研究一边回测就像酒驾。**研究完了再回测,不要回测中调整研究**。
3. **第三定律**: 报告任何回测结果时,**必须同时报告所有尝试过的试验数**——否则无法估计 false discovery 概率。

### 元策略范式 (Meta-Strategy Paradigm)

放弃"50 个 PhD 各自单独造车"的西西弗斯模式,改用"研究工厂"模式:每个量化分析师专注流水线某一环节,但保持对全流程的整体认知。

## 阅读建议

1. **首次通读**: 按章节顺序,每章先读中文笔记,再翻原书对应章节,最后跑配套 Python 演示。
2. **回看时**: 直接看章节末的"本章核心要点",查漏补缺。
3. **遇到具体问题**: 用上面的"十大陷阱表"反查所属章节。
4. **代码层面**: 配套 `python_demos/` 中的脚本均可直接运行,作为 minimal reference 实现。
