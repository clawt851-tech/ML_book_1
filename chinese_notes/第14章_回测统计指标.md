# 第 14 章 回测统计指标

## 14.1 写作动机

无论用什么回测范式(historical / CV / synthetic),都需要按统一指标报告结果以便投资者比较策略。一些来自 GIPS,但 ML 策略需要专门指标。

## 14.2 统计指标分类

回测统计可分为:
- 一般特征 (general characteristics)
- 性能 (performance)
- Runs / drawdowns
- Implementation shortfall
- 收益/风险效率
- 分类得分
- 归因

## 14.3 一般特征

| 指标 | 含义 |
|------|------|
| Time range | 起止日期(应包含多种制度) |
| Average AUM | 平均管理资产 |
| Capacity | 仍能保证目标风险调整收益的最大 AUM |
| Leverage | 杠杆 = 平均美元仓位 / 平均 AUM |
| Max position | 最大美元仓位(应接近 AUM,不依赖极端事件) |
| Ratio of longs | 多头比例(market-neutral 应 ≈ 0.5) |
| Frequency of bets | 每年下注数(同侧持仓算同一次) |
| Holding period | 平均持仓时长 |
| Annualized turnover | 年换手率 |
| Correlation to underlying | 与底层资产相关性 |

## 14.4 性能

PnL、做多 PnL、年化收益、命中率、命中/未命中平均收益。

### 14.4.1 时间加权收益率 (TWRR, GIPS 标准)

$$r_{i,t} = \pi_{i,t} / K_{i,t}, \quad \varphi_{i,T} = \prod_{t=1}^{T}(1 + r_{i,t})$$

## 14.5 Runs

策略收益往往不是 IID,呈现"连续同号 run"。

### 14.5.1 收益集中度 (HHI)

正收益集中度:

$$h^+ = \frac{\sum_t (w_t^+)^2 - \|w^+\|^{-1}}{1 - \|w^+\|^{-1}}, \quad w_t^+ = \frac{r_t^+}{\sum_t r_t^+}$$

(对负收益类似)

**理想策略:**
- 高 SR
- 高下注频率
- 高命中率
- 低 $h^+$(无右尾极端)、低 $h^-$(无左尾极端)
- 低 $h[t]$(下注不集中在某些时段)

### 14.5.2 回撤 (DD) 与水下时间 (TuW)

DD = 两个连续 high-watermark 间最大跌幅;TuW = 从 HWM 到再次创 HWM 的时间。

### 14.5.3 95-percentile DD/TuW: 用分位数比单点最大值更稳健。

## 14.7 效率指标

### 14.7.1 Sharpe Ratio

$$SR = \mu / \sigma$$

### 14.7.2 概率 Sharpe Ratio (PSR)

修正短样本期 + 偏度/峰度引起的虚高:

$$\widehat{PSR}[SR^*] = Z\left[\frac{(\widehat{SR} - SR^*)\sqrt{T - 1}}{\sqrt{1 - \hat{\gamma}_3 \widehat{SR} + \frac{\hat{\gamma}_4 - 1}{4}\widehat{SR}^2}}\right]$$

PSR > 0.95 表示在 5% 显著性下击败基准 SR^*。

### 14.7.3 Deflated Sharpe Ratio (DSR) ★

PSR + 多重检验修正——基准 SR* 由实际试验数 N 决定:

$$SR^* = \sqrt{V[\{\widehat{SR}_n\}]} \left((1-\gamma)Z^{-1}\left[1-\frac{1}{N}\right] + \gamma Z^{-1}\left[1-\frac{1}{Ne}\right]\right)$$

$\gamma$ = Euler-Mascheroni 常数 ≈ 0.5772。N 越多 → SR* 越高 → 越难证明非偶然。

### Marcos 第三回测定律

> "Every backtest result must be reported in conjunction with all the trials involved in its production. Absent that information, it is impossible to assess the backtest's 'false discovery' probability."

## 14.8 分类得分

适用于元标注策略(第 3 章):

- Accuracy = (TP+TN)/(TP+TN+FP+FN)
- Precision = TP/(TP+FP)
- Recall = TP/(TP+FN)
- $F_1 = 2 \cdot \mathrm{precision} \cdot \mathrm{recall} / (\mathrm{precision} + \mathrm{recall})$
- Negative log-loss(第 9 章): 考虑预测概率,优于 accuracy。

## 14.9 归因

把 PnL 拆解到 duration、credit、liquidity、currency、issuer 等风险类。Barra 多因子模型最常用。

## 14.11 本章核心要点

1. 报告策略时使用一组完整指标,不只是 SR。
2. **始终报告进行的试验次数**,否则无法评估假阳概率。
3. 用 **DSR**(Deflated Sharpe Ratio)纠正多重检验偏差。
4. 用 PSR 纠正短样本/偏度/峰度。
5. HHI 衡量集中度,理想策略集中度应低。
