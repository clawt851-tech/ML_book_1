# 第 17 章 结构性突变

## 17.1 写作动机

ML 投资策略最理想的入场时机是"多个因素汇聚"——而**制度切换 (regime change)** 就是这种汇聚的典型例子。例如均值回归市场切换为动量市场时,大多数参与者会措手不及——他们犯下损失,意识到错误时已经太晚。

结构性突变伴随着**最佳的风险/回报机会**。本章介绍量化结构突变可能性的方法。

## 17.2 结构突变检验类型

| 类别 | 含义 |
|------|------|
| **CUSUM 检验** | 累计预测误差是否显著偏离白噪声 |
| **爆炸性检验 (Explosiveness)** | 过程是否表现指数增长/塌陷,与平稳相悖 |
| ↳ 右尾单位根检验 | 假设 AR 形式,检测指数增长 |
| ↳ Sub/Super-martingale 检验 | 多种函数形式下检测指数趋势 |

## 17.3 CUSUM 检验

### 17.3.1 Brown-Durbin-Evans CUSUM(基于递归残差)

设特征 $x_t$ 预测 $y_t$,模型 $y_t = \beta_t' x_t + \varepsilon_t$。用递归最小二乘 (RLS) 在子样本 $[1, k+1], \dots, [1, T]$ 上估计 $\hat{\beta}$。计算标准化的递归残差,累计统计量 $S_t \sim N[0, t-k-1]$ under $H_0$。

### 17.3.2 Chu-Stinchcombe-White CUSUM(基于水平)

简化:假设无变化 $H_0: \beta_t = 0$。直接对 log-price $y_t$ 计算:

$$S_{n,t} = (y_t - y_n)\,\hat{\sigma}_t\sqrt{t-n}^{-1}$$

时间相关临界值 $c_a[n, t] = \sqrt{b_\alpha + \log(t-n)}$,$b_{0.05} \approx 4.6$。

## 17.4 爆炸性检验

### 17.4.1 Chow-type Dickey-Fuller (CDF)

H0: 随机游走;H1: 在 $\tau^* T$ 时刻切换为爆炸过程。问题:断点 $\tau^*$ 通常未知。Andrews [1993] 的解法:在所有可能的 $\tau^*$ 中取最大 t 统计量 → SDFC 统计量。

### 17.4.2 Supremum Augmented Dickey-Fuller (SADF) ★

Phillips-Wu-Yu [2011] 提出。在每个时间点 $t$ 上,扫描所有可能的起始点 $t_0 \in [1, t-\tau]$,拟合 ADF 回归并取最大 t 值:

$$SADF_t = \sup_{t_0 \in [1, t-\tau]} \left\{\frac{\hat{\beta}_{t_0,t}}{\hat{\sigma}_{\beta_{t_0,t}}}\right\}$$

**SADF 在泡沫期会急剧上升,泡沫破裂回归低水平。**

#### 实战要点
- **用 log-price** 而不是原始 price(原始 price 假设方差不变,与现实矛盾)。
- 计算复杂度 $O(n^2)$——一条完整 SADF 序列对 SP500 dollar bar 需要 ~2 PFLOPs,需要 HPC。
- 改进:Quantile ADF (QADF, $q=0.95$) 减小对极端样本的敏感性;Conditional ADF (CADF) 用条件矩。

### 17.4.3 Sub/Super-Martingale 检验 (SMT)

不依赖 ADF 形式,允许多种趋势形式:

| 形式 | 方程 |
|------|------|
| Poly1 | $y_t = \alpha + \gamma t + \beta t^2 + \varepsilon_t$ |
| Poly2 | $\log y_t = \alpha + \gamma t + \beta t^2 + \varepsilon_t$ |
| Exp | $y_t = \alpha e^{\beta t} + \varepsilon_t$ |
| Power | $y_t = \alpha t^\beta + \varepsilon_t$ |

$$SMT_t = \sup_{t_0} \left\{\frac{|\hat{\beta}_{t_0, t}|}{\hat{\sigma}_{\beta_{t_0,t}} (t-t_0)^\varphi}\right\}$$

调节 $\varphi \in [0, 1]$ 偏好长期或短期泡沫。

## 17.6 本章核心要点

1. 结构突变 = ML 策略最赚钱的时机。
2. **CUSUM** 适合早期预警,简单;**SADF** 适合泡沫检测,需要 HPC。
3. 用 **log-price**,不用 raw price。
4. 在 ML 特征矩阵中加入 SADF/SMT,可显著提升对制度切换的捕捉能力。
